Thanks for buy this asset.

To get same quality like on screenshoots, set quality to highest and make sure that in "Edit->Project setting->Player" Color Space = Linear!!!

Asset has been tested also on Unity 5.6 version.

========================================================================

In Unity 5.5+ "Studio" scene using:

-Legacy Cinematic Image Effects (Unity 5.5 or later) Look at asset store.
-Frame Capture github url: https://github.com/unity3d-jp/FrameCapturer

========================================================================

In Unity 2017+ "Studio" scene issues:

1) PostProcess Effects was disabled;
2) Steam has been disabled, to get it work follow next steps (or delete it):

1. Asset Store->Import package->ParticleSystems;
2. Download Unity PostProcessing, url: https://github.com/Unity-Technologies/PostProcessing
3. Enable GameObject Steam;
4. Enable Post Processing Behavior script on Main Camera and look at settings in PDF file.

(-Frame Capture github url: https://github.com/unity3d-jp/FrameCapturer)

Also included PDF file with setting for Legacy Cinematic Emage Effects (Unity 5.5+).
